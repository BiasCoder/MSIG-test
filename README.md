# MSIG-test

A JavaScript coding exercise project with utility functions and Jest-based tests.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or later)
- npm

### Installation

```bash
npm install
```

### Running Tests

```bash
npm test
```

## Project Structure

```
├── src/
│   └── index.js       # Utility function implementations
├── tests/
│   └── index.test.js  # Jest test suite
├── package.json
└── README.md
```

## Functions

| Function | Description |
|---|---|
| `greet(name)` | Returns a greeting string for the given name |
| `add(a, b)` | Returns the sum of two numbers |
| `reverseString(str)` | Returns the reverse of a string |
| `isPrime(n)` | Returns `true` if `n` is a prime number |
| `fibonacci(n)` | Returns the nth Fibonacci number (0-indexed) |
